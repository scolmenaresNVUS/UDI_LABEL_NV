## CRITICAL LABEL FORMAT RULE

This system has TWO label modes with DIFFERENT data fields:

**LOT MODE (Batch/Lot Number):**
- Encodes: GTIN (01) + Lot Number (10) + Manufacturing Date (11)
- Does NOT include Expiration Date вЂ” neither in the DataMatrix barcode nor in the HRI text
- HRI line order: GTIN в†’ Lot в†’ Manufacturing Date
- Example HRI:
  ```
  (01) 00850008393006
  (10) MG96CI-260116
  (11) 260116
  ```

**SERIAL MODE (Serial Number):**
- Encodes: GTIN (01) + Serial Number (21) + Manufacturing Date (11) + Expiration Date (17)
- INCLUDES Expiration Date in both the DataMatrix barcode and HRI text
- HRI line order: GTIN в†’ Serial в†’ Manufacturing Date в†’ Expiration Date
- Example HRI:
  ```
  (01) 00850008393006
  (21) BWIII2026-1234
  (11) 260218
  (17) 330218
  ```

**HRI LINE ORDER is: GTIN first, then the identifier (lot or serial), then date(s).** This order must be consistent in the HRI text display, the preview, and the printed label. The DataMatrix barcode encoding order may differ internally for optimal encoding (fixed-length AIs first), but the PRINTED human-readable text always follows the order above.

This distinction affects EVERY part of the system: form fields shown, GS1 encoding, ZPL generation, and HRI text rendering.

### JSON Store Service

Create `server/src/services/jsonStore.ts` вЂ” a generic, reusable, concurrency-safe JSON file store:

```typescript
import fs from 'fs/promises';
import path from 'path';
import lockfile from 'proper-lockfile';

const DATA_DIR = path.join(process.cwd(), 'data');

/**
 * Generic JSON file store with file-locking for safe concurrent access.
 *
 * Usage:
 *   const store = new JsonStore<User>('users.json');
 *   const users = await store.readAll();
 *   await store.write(updatedUsers);
 *   await store.append(newUser);          // adds one item
 *   await store.updateById(id, partial);  // updates one item by id
 *   await store.deleteById(id);           // removes one item by id
 */
export class JsonStore<T extends { id: string }> {
  private filePath: string;

  constructor(filename: string) {
    this.filePath = path.join(DATA_DIR, filename);
  }

  /**
   * Ensure the data directory and file exist.
   * Call once at app startup.
   */
  async init(): Promise<void> {
    await fs.mkdir(DATA_DIR, { recursive: true });
    try {
      await fs.access(this.filePath);
    } catch {
      await fs.writeFile(this.filePath, '[]', 'utf-8');
    }
  }

  /**
   * Read all records. Uses file lock to prevent reading during writes.
   */
  async readAll(): Promise<T[]> {
    const release = await lockfile.lock(this.filePath, { retries: 5 });
    try {
      const raw = await fs.readFile(this.filePath, 'utf-8');
      return JSON.parse(raw) as T[];
    } finally {
      await release();
    }
  }

  /**
   * Overwrite the entire file with new data.
   */
  async write(data: T[]): Promise<void> {
    const release = await lockfile.lock(this.filePath, { retries: 5 });
    try {
      await fs.writeFile(this.filePath, JSON.stringify(data, null, 2), 'utf-8');
    } finally {
      await release();
    }
  }

  /**
   * Append a single record.
   */
  async append(item: T): Promise<void> {
    const all = await this.readAll();
    all.push(item);
    await this.write(all);
  }

  /**
   * Find by ID.
   */
  async findById(id: string): Promise<T | undefined> {
    const all = await this.readAll();
    return all.find(item => item.id === id);
  }

  /**
   * Update a record by ID (partial update).
   */
  async updateById(id: string, partial: Partial<T>): Promise<T | null> {
    const all = await this.readAll();
    const index = all.findIndex(item => item.id === id);
    if (index === -1) return null;
    all[index] = { ...all[index], ...partial, id }; // preserve ID
    await this.write(all);
    return all[index];
  }

  /**
   * Delete a record by ID.
   */
  async deleteById(id: string): Promise<boolean> {
    const all = await this.readAll();
    const filtered = all.filter(item => item.id !== id);
    if (filtered.length === all.length) return false;
    await this.write(filtered);
    return true;
  }
}
```
